# port_me

Portfolio pessoal desenvolvido com Next.js 15, React 19 e Tailwind CSS 4.

## Stack

- **Framework:** Next.js 15 (App Router)
- **UI:** React 19 + Tailwind CSS 4
- **Linguagem:** TypeScript

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
